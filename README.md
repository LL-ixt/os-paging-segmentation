# Minh họa Quản lý bộ nhớ trong Hệ điều hành

Chương trình minh họa cho các giải thuật quản lý bộ nhớ theo phương pháp phân trang, phân đoạn.

## Cài đặt

1. **Compile:**
   
    ```bash
    javac *.java

1. **Chạy file `Main.java`:**
   
    ```bash
    java Main
